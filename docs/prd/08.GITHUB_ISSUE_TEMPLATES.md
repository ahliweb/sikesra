# GitHub Issue Templates — SIKESRA

Dokumen ini berisi template standar untuk issue GitHub SIKESRA yang siap dieksekusi AI model junior.

---

## Template 1: Feature Issue (AI-Ready)

```markdown
## Context Capsule

> Baca HANYA ini sebelum mulai. Jangan buka dokumen lain kecuali yang disebutkan di "References".

**Plugin**: `@ahliweb/awcms-sikesra`  
**Dir**: `awcmsmicro-dev/packages/plugins/awcms-sikesra/`  
**Platform**: AWCMS-Micro / Cloudflare Workers / D1  
**Invariants** (lihat `03.PLUGIN_ARCHITECTURE.md` § 8 untuk daftar penyimpangan lengkap):
- Storage: `ctx.storage.<collectionName>` — collections berindeks (lihat `04.DATABASE_SCHEMA.md`), BUKAN tabel SQL `sikesra_*`
- Plugin ID nyata: `awcms-sikesra` (BUKAN `awcms-micro-sikesra`)
- Audit wajib setiap mutasi: `createAuditRecord()` + `appendAuditEvent()` dari `src/runtime.ts` (BUKAN `src/audit.ts`, yang dead code)
- RBAC/ABAC: belum ada enforcement server-side nyata di route mutasi — baca `10.SECURITY_AND_PRIVACY_CHECKLIST.md` § 0 sebelum menulis kode yang mengasumsikan otorisasi sudah berfungsi
- String permission nyata didefinisikan inline di manifest `runtime.ts`, BUKAN di `src/permissions.ts` (tidak sinkron, hanya dipakai test)
- UI: Kumo only, Lingui i18n, logical Tailwind (ms-/me-/ps-/pe-)
```
